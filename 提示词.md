# 开发规范（必须严格遵守）

## 1. 目录与文件结构
- **UI 界面**：统一放在 `UI/` 目录下。
  - 完整页面：放在 `UI/Pages/`
  - 页面组件/小组件：放在 `UI/Components/`
- **数据模型**：所有对象模型（Model）放在 `models/` 目录下。
- **配置与常量**：后端接口地址、常量定义统一放在 `config/Constants` 中。
- **主题资源**：颜色与尺寸定义放在 `theme/Colors` 和 `theme/Dimens` 中。

## 2. UI 样式与主题规范
- **间距 (Spacing/Padding/Margin)**：
  - 应用中**所有**间距（包括图标间距、按钮间距、图文间距、布局边距、列表条目间隔等）统一使用 `theme/Dimens.middleMargin`。
- **卡片式模块**：
  - 应用中除了ChatPage页面外，其他所有页面模块都是使用卡片式，
  - 卡片圆角大小： `Dimens.borderRadius`
  - 卡片背景色：`Colors.whiteColor`
  - 卡片内边距：`Dimens.middleMargin`
- **颜色 (Colors)**：
  - 页面背景色：`Colors.pageBackgroundColor`
  - 应用主色调：`Colors.primaryColor`
  - 文字颜色：白色使用 `Colors.whiteColor`，常规/边框灰色使用 `Colors.grayColor`
  - 警告/删除色：`Colors.warnColor`
- **按钮 (Buttons)**：
  - 高度：统一使用 `Dimens.btnHeight`
  - 圆角：`Dimens.btnHeight / 2`
  - **确定/激活按钮**：背景 `Colors.primaryColor`，文字 `Colors.whiteColor`
  - **取消按钮**：背景透明，边框和文字 `Colors.grayColor`
  - **禁用状态**：背景 `Colors.grayColor`
  - **删除按钮**：背景 `Colors.warnColor`
- **输入框与卡片 (Inputs & Cards)**：
  - 单行搜索框/输入框高度：`Dimens.inputHeight`
  - 输入框圆角：`Dimens.inputHeight / 2`
  - 卡片式矩形圆角：`Dimens.borderRadius`
- **图标 (Icons)**：
  - 小图标尺寸：`Dimens.smallAvater`
  - 中图标尺寸：`Dimens.middleAvater`
  - 大图标尺寸：`Dimens.bigAvater`
  - 图标透明度：统一设置为 `0.5`
- **字体 (Typography)**：
  - 正常字体：`Dimens.normalFont`
  - 主标题：`Dimens.middleFont`
  - 大标题：`Dimens.bigFont`

## 3. 数据交互与接口规范
- **统一返回格式**：所有后端接口返回数据格式为 `Models/BaseResponse`。
  - `data`: 后端返回的实际数据（JSON、数组、字符串、数字等）。
  - `total`: 仅分页加载接口返回总条数，其他接口为 `null`。
  - `msg`: 服务器返回信息（通常在失败时返回）。
  - `status`: 字符串类型，`"SUCCESS"` 表示调用成功，其他值表示失败。
- **接口定义**：
  - 所有接口路径定义在 `config/Constants` 的 `APIEndpoint` 中。
  - **请求头**：除登录接口外，所有请求必须在 Header 中添加 `Authorization: Bearer <token>`。
